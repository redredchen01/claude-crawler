## v0.5 規劃（新增 3 個高頻模組 + 5 個子命令）

### 模組 A: `llm` — LLM 工作流加速

**觸發點**：每次 prompt 工程/DSPy 開發都需要手動計算 token、管理 API key、測試 prompt 變體

| 子命令 | 功能 | 優先級 |
|--------|------|--------|
| `ydk llm token` | 估算 prompt/response token 數 | ⭐⭐⭐ |
| `ydk llm split` | 按 token 限制分割長文本 | ⭐⭐ |
| `ydk llm test` | 批量測試 prompt 變體（乾跑，不計費） | ⭐⭐ |
| `ydk llm cache` | 本地 API 緩存管理（避免重複調用） | ⭐⭐ |

**程式碼架構**：
```python
# src/ydk/commands/llm.py
- TokenCounter (支持 Claude / OpenAI / Llama 三種計數方式)
- PromptSplitter (尊重 token 邊界，不截斷)
- PromptTester (dry-run mode，模擬 API 調用)
- CacheManager (SQLite 本地存儲)
```

**測試清單**：
```bash
test_token_counter_claude()
test_token_counter_openai()
test_split_preserves_semantics()
test_cache_hit_ratio()
test_test_dry_run()
```

### 模組 B: `api` — API 呼叫工具鏈

**觸發點**：GWX 的 API 集成需要重試、速率限制、錯誤恢復，現在靠手寫

| 子命令 | 功能 | 優先級 |
|--------|------|--------|
| `ydk api build` | 快速構建 HTTP 請求（支持 headers/auth/body） | ⭐⭐⭐ |
| `ydk api test` | 單次請求測試（含重試策略） | ⭐⭐⭐ |
| `ydk api batch` | 批量 API 調用（並行 + 速率限制） | ⭐⭐ |
| `ydk api docs` | 從 curl 反向生成 OpenAPI 片段 | ⭐ |

**程式碼架構**：
```python
# src/ydk/commands/api.py
- RequestBuilder (fluent API 風格)
- RetryStrategy (exponential backoff + jitter)
- RateLimiter (令牌桶算法)
- BatchRunner (concurrent.futures 並行調用)
```

**測試清單**：
```bash
test_build_with_headers()
test_retry_on_429()
test_rate_limiter_respects_quota()
test_batch_parallel_execution()
```

### 模組 C: `storage` — 本地資料管理

**觸發點**：
- Obsidian 知識庫備份和同步
- TG Bot 的 session 存儲
- GWX 的臨時資料（競品數據、爬蟲結果）

| 子命令 | 功能 | 優先級 |
|--------|------|--------|
| `ydk storage backup` | 增量備份到 S3/本地/Dropbox | ⭐⭐⭐ |
| `ydk storage sync` | 雙向同步（Obsidian vault） | ⭐⭐ |
| `ydk storage db` | SQLite 快速操作（create/insert/query） | ⭐⭐ |
| `ydk storage cache` | 緩存預熱和過期策略 | ⭐ |

**程式碼架構**：
```python
# src/ydk/commands/storage.py
- BackupManager (rsync-style 增量)
- SyncEngine (two-way sync，衝突解決)
- DatabaseHelper (ORM-less 輕量級 CRUD)
- CacheLifecycle (TTL + LRU 淘汰)
```

**測試清單**：
```bash
test_backup_incremental()
test_sync_detects_conflicts()
test_db_transaction_rollback()
test_cache_ttl_expiry()
```
